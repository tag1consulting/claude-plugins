# tag1consulting — Claude Code Plugin Marketplace

A Claude Code plugin marketplace for plugins published by [Tag1 Consulting](https://tag1consulting.com).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [comprehensive-review](https://github.com/tag1consulting/claude-comprehensive-review) | CodeRabbit-style PR review using parallel specialized agents |

## Installation

Install any plugin inside Claude Code:

```
/plugins install <plugin-name>@tag1consulting
```

For example:

```
/plugins install comprehensive-review@tag1consulting
```
